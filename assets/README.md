# 图标说明

此扩展需要一个 `icon.png` 文件作为 Raycast 扩展的图标。

## 要求

- **尺寸**: 512x512 像素(推荐)
- **格式**: PNG
- **文件名**: `icon.png`
- **位置**: 此目录(`assets/`)

## 设计建议

图标应该代表 fnm (Fast Node Manager) 和 Node.js:

- 可以使用 Node.js 的标志性绿色
- 可以包含 "fnm" 字样或首字母
- 可以融入速度/快速的视觉元素(如闪电、箭头等)
- 保持简洁,在小尺寸下也要清晰可辨

## 创建图标

您可以:

1. 使用设计工具(如 Figma, Sketch, Photoshop)创建
2. 使用在线图标生成器
3. 从 fnm 的官方资源中获取(如果有)

创建完成后,将文件命名为 `icon.png` 并放置在此目录中。

## 临时解决方案

在没有自定义图标之前,您可以使用任何 512x512 的 PNG 图片作为占位符。
